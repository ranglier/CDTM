# Roadmap

## Phase 0 : documentation et modele de donnees

- Poser la structure du projet.
- Documenter les besoins, nomenclatures et schemas de base.

## Phase 1 : carte statique interactive

- Afficher une carte simple avec fond et navigation.
- Tester une integration cartographique minimale.

## Phase 2 : chargement GeoJSON

- Charger les premieres geometries de cases depuis des exports prepares.
- Valider la separation entre source geographique et donnees metier.

## Phase 3 : popup et filtres

- Ajouter les popups de consultation.
- Ajouter des filtres simples sur les proprietes des cases.

## Phase 4 : interface staff

- Introduire une vue ou un mode staff.
- Charger des donnees internes separees si necessaire.

## Phase 5 : edition collaborative

- Explorer des mecanismes de mise a jour encadree des donnees.
- Definir les regles de contribution et de validation.

## Phase 6 : historique temporel

- Ajouter la gestion des evolutions dans le temps.
- Prevoir une lecture chronologique des changements de controle.

## Note

La roadmap est indicative et peut etre reordonnee selon les besoins du projet.
