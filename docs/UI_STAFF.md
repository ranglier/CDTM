# Interface staff

## Objectif

La vue staff permet de consulter et, plus tard, modifier les informations privées ou administratives de la carte.

## Informations visibles staff

- toutes les informations publiques ;
- `note_staff` ;
- calques privés ;
- statuts incertains ;
- zones contestées ;
- informations préparatoires non validées.

## Fonctionnalités futures

- édition des champs d’une case ;
- import / export de données ;
- filtres avancés ;
- historique de contrôle ;
- affichage des localités ;
- validation ou verrouillage de certaines données.

## Sécurité des données

Les données staff ne doivent pas être incluses dans les exports publics.

À terme, prévoir une séparation claire entre :

- fichiers publics ;
- fichiers staff ;
- données secrètes ou incertaines.

## Filtres staff envisagés

- faction ;
- contrôleur ;
- type de contrôle ;
- race ;
- terrain ;
- présence de note staff ;
- statut incertain ;
- cases contestées ;
- zones occupées.
